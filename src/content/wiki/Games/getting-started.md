# Getting Started with Games Section

This section will be used for notes on game prototypes, emulators, and retro consoles.

## What Can Be Added

- Description of the game itself or genre.
- List of resources (ROMs, emulators, emulator settings).
- Personal progress or test results.

## Next Steps

1. Create subfolders by project name, e.g., `Games/NES/`.
2. Add separate notes with instructions and screenshots.
3. Link entries with applications (e.g., Minesweeper or Doom) through "My Projects".

# Начало работы с разделом Games

Этот раздел будет использоваться для заметок об игровых прототипах, эмуляторах и ретро‑консолях.

## Что можно добавить

- Описание самой игры или жанра.
- Список ресурсов (ромы, эмуляторы, эмуляторные настройки).
- Личный прогресс или результаты тестов.

## Дальнейшие шаги

1. Создайте подпапки по названию проекта, например `Games/NES/`.
2. Добавьте отдельные заметки с инструкциями и скриншотами.
3. Свяжите записи с приложениями (например, Minesweeper или Doom) через «Мои проекты».
