# Getting Started with React

This section is intended for notes on components, hooks, and working patterns.

## Minimal Project

```bash
npm create vite@latest my-react-app -- --template react-ts
cd my-react-app
npm install
npm run dev
```

## What to Describe Next

- Testing (Vitest/Jest) and UI stories.
- Architecture (feature-sliced, clean architecture).
- Optimizations (memo, Suspense, server components).

Use this folder for notes on specific libraries: react-router, zustand, tanstack query, etc.
