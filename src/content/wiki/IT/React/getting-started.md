# Начало работы с React

Раздел предназначен для конспектов по компонентам, хукам и рабочим паттернам.

## Минимальный проект

```bash
npm create vite@latest my-react-app -- --template react-ts
cd my-react-app
npm install
npm run dev
```

## Что описать дальше

- Тестирование (Vitest/Jest) и истории UI.
- Архитектуру (feature-sliced, clean architecture).
- Оптимизации (memo, Suspense, серверные компоненты).

Используйте эту папку для заметок по конкретным библиотекам: react-router, zustand, tanstack query и т.д.
