# Getting Started with Proxmox

This document will be the entry point for notes on home hypervisor and container management.

## What to Prepare

1. **Hardware.** Specify server/NUC parameters and memory and disk requirements.
2. **Network Settings.** Describe VLAN, bridge, and access rules.
3. **Storage.** Add steps for configuring ZFS/LVM and backups.

## Further Notes

- Create files like `provisioning.md`, `upgrade-notes.md`.
- Record useful commands (e.g., `pct`, `qm`, `pvesh`).
- Add checklists for deploying virtual machines and LXC.

# Начало работы с Proxmox

Этот документ станет точкой входа для заметок по домашнему гипервизору и управлению контейнерами.

## Что подготовить

1. **Железо.** Укажите параметры сервера/NUC и требования по памяти и дискам.
2. **Сетевые настройки.** Опишите VLAN, bridge и правила доступа.
3. **Хранилище.** Добавьте шаги по настройке ZFS/LVM и бэкапов.

## Дальнейшие заметки

- Создайте файлы вида `provisioning.md`, `upgrade-notes.md`.
- Зафиксируйте полезные команды (например, `pct`, `qm`, `pvesh`).
- Добавьте чек-листы развёртывания виртуальных машин и LXC.
