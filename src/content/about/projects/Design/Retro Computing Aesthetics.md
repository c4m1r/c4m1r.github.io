# Retro Computing Aesthetics

Research and recreation of retro computer aesthetics. From Windows 98 to Mac OS Classic.

## Studied Systems

- **Windows 98/XP**: Classic Microsoft design
- **Mac OS Classic**: Platinum theme and Aqua
- **BeOS**: Unique BeOS aesthetics
- **AmigaOS**: Workbench and its features

## Application

Using retro aesthetics in modern projects to create a nostalgic atmosphere.

## Tools

- CSS for recreating styles
- Pixel art tools
- Historical research
- UI component libraries

## Result

A library of components in the style of various retro systems, ready for use in web projects.

# Retro Computing Aesthetics

Исследование и воссоздание эстетики ретро-компьютеров. От Windows 98 до Mac OS Classic.

## Исследованные системы

- **Windows 98/XP**: Классический дизайн Microsoft
- **Mac OS Classic**: Платиновая тема и Aqua
- **BeOS**: Уникальная эстетика BeOS
- **AmigaOS**: Workbench и его особенности

## Применение

Использование ретро-эстетики в современных проектах для создания ностальгической атмосферы.

## Инструменты

- CSS для воссоздания стилей
- Pixel art tools
- Historical research
- UI component libraries

## Результат

Библиотека компонентов в стиле различных ретро-систем, готовых к использованию в веб-проектах.
