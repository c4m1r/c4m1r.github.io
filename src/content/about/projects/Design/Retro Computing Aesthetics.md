# Retro Computing Aesthetics

Исследование и воссоздание эстетики ретро-компьютеров. От Windows 98 до Mac OS Classic.

## Исследованные системы

- **Windows 98/XP**: Классический дизайн Microsoft
- **Mac OS Classic**: Платиновая тема и Aqua
- **BeOS**: Уникальная эстетика BeOS
- **AmigaOS**: Workbench и его особенности

## Применение

Использование ретро-эстетики в современных проектах для создания ностальгической атмосферы.

## Инструменты

- CSS для воссоздания стилей
- Pixel art tools
- Historical research
- UI component libraries

## Результат

Библиотека компонентов в стиле различных ретро-систем, готовых к использованию в веб-проектах.
