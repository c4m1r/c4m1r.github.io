# Portfolio Website

Современный портфолио-сайт с множественными темами и режимами отображения. От классического GRUB до Windows XP и современного веб-дизайна.

## Особенности дизайна

- **Multiple Themes**: Более 8 уникальных тем (Frutiger Aero, Vaporwave, Cyberpunk, Skeuomorphism, PCB)
- **Responsive Design**: Адаптивный дизайн для всех устройств
- **Weather Effects**: Анимированные погодные эффекты (снег, дождь, фейерверки)
- **Smooth Animations**: Плавные переходы и анимации

## Технологии

- React + TypeScript
- Tailwind CSS
- Vite
- Canvas API для эффектов
- CSS Variables для тем

## Концепция

Сочетание ностальгии и современных веб-технологий. Каждая тема - это отдельный мир со своей атмосферой.
