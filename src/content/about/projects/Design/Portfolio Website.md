# Portfolio Website

A modern portfolio website with multiple themes and display modes. From classic GRUB to Windows XP and modern web design.

## Design Features

- **Multiple Themes**: Over 8 unique themes (Frutiger Aero, Vaporwave, Cyberpunk, Skeuomorphism, PCB)
- **Responsive Design**: Adaptive design for all devices
- **Weather Effects**: Animated weather effects (snow, rain, fireworks)
- **Smooth Animations**: Smooth transitions and animations

## Technologies

- React + TypeScript
- Tailwind CSS
- Vite
- Canvas API for effects
- CSS Variables for themes

## Concept

A combination of nostalgia and modern web technologies. Each theme is a separate world with its own atmosphere.

# Portfolio Website

Современный портфолио-сайт с множественными темами и режимами отображения. От классического GRUB до Windows XP и современного веб-дизайна.

## Особенности дизайна

- **Multiple Themes**: Более 8 уникальных тем (Frutiger Aero, Vaporwave, Cyberpunk, Skeuomorphism, PCB)
- **Responsive Design**: Адаптивный дизайн для всех устройств
- **Weather Effects**: Анимированные погодные эффекты (снег, дождь, фейерверки)
- **Smooth Animations**: Плавные переходы и анимации

## Технологии

- React + TypeScript
- Tailwind CSS
- Vite
- Canvas API для эффектов
- CSS Variables для тем

## Концепция

Сочетание ностальгии и современных веб-технологий. Каждая тема - это отдельный мир со своей атмосферой.
