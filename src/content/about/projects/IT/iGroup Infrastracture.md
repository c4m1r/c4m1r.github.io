# iGroup Infrastructure

A comprehensive project for building IT infrastructure for iGroup. Includes server setup, network infrastructure, security systems, and monitoring.

## Main Components

- **Server Infrastructure**: Setup and administration of Linux-based servers
- **Network Architecture**: Design and implementation of network topology
- **Security Systems**: Configuration of firewalls, VPN, intrusion detection systems
- **Monitoring**: Implementation of monitoring and logging systems

## Technologies

- Linux (Ubuntu Server, CentOS)
- Docker & Kubernetes
- Nginx, Apache
- PostgreSQL, Redis
- Prometheus, Grafana
- Ansible, Terraform

## Results

Created a reliable and scalable IT infrastructure ensuring uninterrupted operation of all company services.

# iGroup Infrastructure

Комплексный проект по построению IT-инфраструктуры для iGroup. Включает в себя настройку серверов, сетевой инфраструктуры, систем безопасности и мониторинга.

## Основные компоненты

- **Серверная инфраструктура**: Настройка и администрирование серверов на базе Linux
- **Сетевая архитектура**: Проектирование и внедрение сетевой топологии
- **Системы безопасности**: Настройка файрволов, VPN, системы обнаружения вторжений
- **Мониторинг**: Внедрение систем мониторинга и логирования

## Технологии

- Linux (Ubuntu Server, CentOS)
- Docker & Kubernetes
- Nginx, Apache
- PostgreSQL, Redis
- Prometheus, Grafana
- Ansible, Terraform

## Результаты

Создана надежная и масштабируемая IT-инфраструктура, обеспечивающая бесперебойную работу всех сервисов компании.
