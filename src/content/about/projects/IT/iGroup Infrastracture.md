# iGroup Infrastructure

Комплексный проект по построению IT-инфраструктуры для iGroup. Включает в себя настройку серверов, сетевой инфраструктуры, систем безопасности и мониторинга.

## Основные компоненты

- **Серверная инфраструктура**: Настройка и администрирование серверов на базе Linux
- **Сетевая архитектура**: Проектирование и внедрение сетевой топологии
- **Системы безопасности**: Настройка файрволов, VPN, системы обнаружения вторжений
- **Мониторинг**: Внедрение систем мониторинга и логирования

## Технологии

- Linux (Ubuntu Server, CentOS)
- Docker & Kubernetes
- Nginx, Apache
- PostgreSQL, Redis
- Prometheus, Grafana
- Ansible, Terraform

## Результаты

Создана надежная и масштабируемая IT-инфраструктура, обеспечивающая бесперебойную работу всех сервисов компании.
