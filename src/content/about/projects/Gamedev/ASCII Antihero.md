# ASCII Antihero

Сюжетная RPG с ASCII-графикой, где игрок берет на себя роль антигероя. Моральные дилеммы, сложные выборы и последствия действий.

## Особенности

- **Story-driven**: Глубокий сюжет с множественными концовками
- **Moral Choices**: Выборы влияют на развитие истории
- **Turn-based Combat**: Тактические бои
- **Character Development**: Прокачка персонажа и навыков

## Технологии

- Python
- ASCII rendering engine
- Dialogue system
- Save/Load system
- Quest management

## Статус

Завершен основной сюжет. Ведется работа над дополнительным контентом.
