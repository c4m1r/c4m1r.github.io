# ASCII Antihero

A story-driven RPG with ASCII graphics where the player takes on the role of an antihero. Moral dilemmas, difficult choices, and consequences of actions.

## Features

- **Story-driven**: Deep storyline with multiple endings
- **Moral Choices**: Choices affect story development
- **Turn-based Combat**: Tactical battles
- **Character Development**: Character and skill progression

## Technologies

- Python
- ASCII rendering engine
- Dialogue system
- Save/Load system
- Quest management

## Status

Main story completed. Working on additional content.
