# ASCII RUN

Ретро-игра в стиле ASCII roguelike с процедурной генерацией уровней. Быстрый геймплей, пермасмерть и классическая ASCII-графика.

## Особенности

- **ASCII Graphics**: Полностью текстовая графика в духе классических roguelike
- **Procedural Levels**: Бесконечная реиграбельность
- **Permadeath**: Каждая смерть - новое начало
- **Fast-paced**: Динамичный геймплей

## Технологии

- Python
- curses/blessed library
- Procedural generation algorithms
- Roguelike mechanics

## Платформы

Windows, Linux, macOS - везде, где есть терминал!
