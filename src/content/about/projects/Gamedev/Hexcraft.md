# Hexcraft

A strategic game with hexagonal grid and procedural world generation. Players build civilizations, explore territories, and interact with other factions.

## Features

- **Hexagonal Grid**: Unique movement and interaction system
- **Procedural Generation**: Each game creates a unique world
- **Resource System**: Complex economic model
- **Diplomacy**: Interaction with AI factions

## Technologies

- Unity Engine
- C#
- Procedural Generation Algorithms
- Pathfinding (A*)
- Custom Hex Grid System

## Status

In active development. Early access release planned for 2025.

# Hexcraft

Стратегическая игра с гексагональной сеткой и процедурной генерацией мира. Игроки строят цивилизацию, исследуют территории и взаимодействуют с другими фракциями.

## Особенности

- **Гексагональная сетка**: Уникальная система передвижения и взаимодействия
- **Процедурная генерация**: Каждая игра создает уникальный мир
- **Система ресурсов**: Сложная экономическая модель
- **Дипломатия**: Взаимодействие с AI-фракциями

## Технологии

- Unity Engine
- C#
- Procedural Generation Algorithms
- Pathfinding (A*)
- Custom Hex Grid System

## Статус

В активной разработке. Планируется релиз в раннем доступе в 2025 году.
