# 법적 고지

**연락처**: [GitHub 프로필](https://github.com/c4m1r)  
**웹사이트**: https://c4m1r.github.io

## 콘텐츠 및 저작권

텍스트, 그래픽, 로고, 이미지 및 소프트웨어를 포함하되 이에 국한되지 않는 이 웹사이트의 모든 콘텐츠는 C4m1r 또는 해당 콘텐츠 공급업체의 재산이며 국제 저작권법의 보호를 받습니다.

### 오픈 소스

이 웹사이트는 오픈 소스 기술을 사용하여 구축되었습니다:
- React
- TypeScript
- Tailwind CSS
- Vite

이 웹사이트의 소스 코드는 MIT 라이선스에 따라 GitHub에서 사용할 수 있습니다.

## 개인정보 보호정책

이 웹사이트는 개인 데이터를 수집하지 않습니다. 호스팅을 위해 GitHub Pages를 사용하며, 익명 사용 통계를 수집할 수 있습니다.

### 쿠키

이 웹사이트는 사용자 기본 설정(테마, 언어, 날씨 효과)을 저장하기 위해 localStorage를 사용합니다. 추적 쿠키는 사용되지 않습니다.

## 면책 조항

이 웹사이트에서 제공되는 정보는 일반적인 정보 제공 목적으로만 제공됩니다. 정보를 최신 상태로 정확하게 유지하기 위해 노력하지만 웹사이트에 포함된 정보의 완전성, 정확성, 신뢰성, 적합성 또는 가용성에 대해 명시적이든 묵시적이든 어떠한 진술이나 보증도 하지 않습니다.

## 외부 링크

이 웹사이트에는 외부 웹사이트에 대한 링크가 포함될 수 있습니다. 이러한 사이트의 콘텐츠와 특성을 제어할 수 없으며 해당 콘텐츠에 대해 책임을 질 수 없습니다.
