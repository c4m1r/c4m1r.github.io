# Система переводов контента

## Обзор

Сайт поддерживает многоязычность для всего контента через систему суффиксов файлов.

## Поддерживаемые языки

- `en` - English (по умолчанию)
- `ru` - Русский
- `fr` - Français
- `es` - Español
- `zh` - 中文
- `ja` - 日本語
- `ko` - 한국어

## Структура файлов

### About контент

Файлы в `src/content/about/`:
- `c4m1r.md` - базовая версия (английский)
- `c4m1r-ru.md` - русская версия
- `c4m1r-fr.md` - французская версия
- `c4m1r-es.md` - испанская версия
- `c4m1r-zh.md` - китайская версия
- `c4m1r-ja.md` - японская версия
- `c4m1r-ko.md` - корейская версия

То же самое для `legal-notice.md`.

### Blog посты

Файлы в `src/content/blog/`:
- `welcome.md` - базовая версия (русский оригинал)
- `welcome-en.md` - английская версия
- `welcome-ru.md` - русская версия (если отличается от базовой)
- `welcome-fr.md` - французская версия
- и т.д.

### Wiki статьи

Файлы в `src/content/wiki/`:
- `IT/React/getting-started.md` - базовая версия
- `IT/React/getting-started-ru.md` - русская версия
- `IT/React/getting-started-en.md` - английская версия
- и т.д.

## Как работает система

1. **Автоматическое определение языка**: Система определяет язык файла по суффиксу `-{lang}` перед `.md`
2. **Fallback на базовую версию**: Если перевода нет, используется базовая версия (без суффикса)
3. **Динамическая загрузка**: Контент загружается в зависимости от выбранного языка пользователя

## Добавление нового перевода

### Для About контента

1. Создайте файл с суффиксом языка: `c4m1r-{lang}.md`
2. Скопируйте структуру из базового файла
3. Переведите контент

### Для Blog постов

1. Создайте файл с суффиксом: `post-name-{lang}.md`
2. Добавьте frontmatter с метаданными
3. Переведите контент
4. Добавьте в конец: `*Эта статья была переведена автоматически.*` (для не-оригинальных языков)

### Для Wiki статей

1. Создайте файл в той же директории: `article-name-{lang}.md`
2. Сохраните структуру категорий
3. Переведите контент

## Функции загрузки

### loadAboutMe(language)
Загружает About me контент для указанного языка.

### loadLegalNotice(language)
Загружает Legal Notice для указанного языка.

### loadBlogPosts(language)
Загружает blog посты для указанного языка.

### loadWikiArticles(category, language)
Загружает wiki статьи для указанной категории и языка.

## Примечания

- Базовые файлы (без суффикса) используются как fallback
- Для автоматически переведенных материалов добавляется уведомление внизу
- Система автоматически перезагружает контент при смене языка
- CV данные хранятся в `cv-data.json` с отдельными секциями для каждого языка

## Тестирование

1. Переключите язык в интерфейсе
2. Проверьте, что контент обновился
3. Убедитесь, что для отсутствующих переводов показывается базовая версия
4. Проверьте, что уведомление о переводе отображается корректно
