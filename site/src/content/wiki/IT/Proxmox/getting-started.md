# Начало работы с Proxmox

Этот документ станет точкой входа для заметок по домашнему гипервизору и управлению контейнерами.

## Что подготовить

1. **Железо.** Укажите параметры сервера/NUC и требования по памяти и дискам.
2. **Сетевые настройки.** Опишите VLAN, bridge и правила доступа.
3. **Хранилище.** Добавьте шаги по настройке ZFS/LVM и бэкапов.

## Дальнейшие заметки

- Создайте файлы вида `provisioning.md`, `upgrade-notes.md`.
- Зафиксируйте полезные команды (например, `pct`, `qm`, `pvesh`).
- Добавьте чек-листы развёртывания виртуальных машин и LXC.
