# Начало работы с Games

Этот раздел будет использоваться для заметок об игровых прототипах, эмуляторах и ретро‑консолях.

## Что можно добавить

- Описание самой игры или жанра.
- Список ресурсов (ромы, эмуляторы, эмуляторные настройки).
- Личный прогресс или результаты тестов.

## Дальнейшие шаги

1. Создайте подпапки по названию проекта, например `Games/NES/`.
2. Добавьте отдельные заметки с инструкциями и скриншотами.
3. Свяжите записи с приложениями (например, Minesweeper или Doom) через «Мои проекты».
